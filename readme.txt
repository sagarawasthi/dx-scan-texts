Displays copied percentage as a result of scanned articles. 

== Description ==

Compare and scan two texts/texts for duplicacy.  Display results in %age. Helps you to remove duplicacy in the content.

== Installation ==

1. Upload the `wp-custom-menu-filter-plugin` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings Â» Dx Scan Texts.
4. Add your original and rewritten Texts/Articles/Posts.
5. Click 'Scan Text' button and you are good to go!
